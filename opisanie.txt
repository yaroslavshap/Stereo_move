1. - Fastapi_pr1 - проект в котором происходит передача стереопар с одного микросервиса на другой с помощью FastApi в виде UploadFiles
2. - Fastapi_pr2 - проект в котором происходит передача стереопар с одного микросервиса на другой с помощью FastApi в виде bytes
3. - GRPC_practic - проект в котором реализованы все 4 способа передачи данных через gRPC (передача файлов и ответ / передача, преобразование, сохранение и ответ)
4. - GRPC_practic_neural - проект в котором все 4 способа передачи данных через gRPC с обученной нейросетью на стороне сервера
5. - GRPC_stream - проект в котором реализованны параллельная подгрузка данных и передача на сервер и возврат ответа (в 2 потока)
6. - GRPC_stream_neural - проект в котором происходит 2 параллельных потока - один подгружает изображения, второй отправляет их на сервер и там нейронка строит карту глубин и возвращает ответ
7. - Pereobuchenie - проект в котором обучение нейронки осуществляется с помощью распределенного обучения
